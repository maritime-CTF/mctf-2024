Submitting Agents

Agents will be submitted via codalab submissions will not be accepted through any other platform. Users should first register for a free codalab account and navigate to the Maritime Capture the Flag competition here. Users should then click on the Participate tab then select submit/view results this is where you should drop your submission zip folder there is a specific formatting you must follow please see the section "formatting your submission" below.

Formatting Your Submission

Export the policies from saved your saved checkpoint:

 1.) python competition_export_policy.py <path_to_saved_checkpoint>

  • Example Usage: python competition_export_policy.py ./ray_test/checkpoint_000001/

  • Outputs two individual saved policy folders into a “/rl_test/policies/” directory

 2.) Select and zip your solution.py and the policy checkpoints your submission uses. DO NOT USE tar, gzip, or any other compressed folder format.

 3.) Submit zip file via An example of how to format a submission entry is given in the zip file at rl_test/example_competition_submission.zip.

The competition code is currently set up to allow for training using RLlib. If you would like to use different reinforcement learning library, contact the competition organizers through challenge.gov and we can work to get your requested dependencies included into the competition.
