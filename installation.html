Setting up a python virtual environment

Using Miniconda (Recomended Method)

 1. Install Miniconda here

 2. Navigate into the git repository in the terminal

 3. Run './setup-conda-env.sh light'(Light installs just the environment) or './setup-conda-env.sh full' (Recomended includes rllib for Deep Reinforcement Leanring)

Congratualations you should now have installed pyquaticus now you can begin training!

Using Python Virutal Environment

 1. In the terminal run python3.10 -m venv <name>

 2. To activate the environment run 'source ./<name>/bin/activate'

 3. Navigate into the pyquaticus git repository in the terminal

 4. To install pyquaticus run 'pip install -e .[torch]'

 5. To install rllib run 'pip install ray[rllib]==2.4.0'

 6. Install gymnasium run 'pip install gymnasium==0.28.1'

Note if you are having trouble installing the packages try installing using 'python -m pip install <package name>'
